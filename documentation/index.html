Documentation

This is the documentation of the Mediated Contestation in Comparative Perspective (MedCon) project.

The documentation is structured into five major sections:

  • Media Selection

    Section Media Selection describes the start of the data collection of the project, where the media to be collected was defined by an expert group in each of the included countries.

  • Text Data Collection

    Section Text Data Collection gives a description of the processes for the data collection.

  • Item Selection

    Section Item Selection describes the topic modeling process and the thematic classification of the plain text data and the manual identification process for non-plain text data.

  • Content Analysis

    Section Content Analysis documents the process and provides the material for reproduction of the content analysis step.

  • List of Files

    List of Files contains a list of all files in the documentation.
