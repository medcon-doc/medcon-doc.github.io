Selection Protocol

In the selection protocol the instructions guiding the manual coding process are documented. Human coders were trained applying the criteria defined in the selection protocol using a test sample. Inter-coder reliabilities are presented in the next section. Next, human coders applied the protocol to the items preselected by the topic modeling process. We used these manual codings for the evaluation of the thematic classification (see section Evaluation).
